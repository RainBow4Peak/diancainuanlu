项目简介：采用“按键+蓝牙”控制，使用了STM32取代传统工艺上的单片机实现自动控温、三定时设置、四种工作模式功能和漏电保护；
核心芯片：STM32F103C8T6
软件：HT-IDE3000V7.96和Keil5、调试使用JLink_Windows_V622d
